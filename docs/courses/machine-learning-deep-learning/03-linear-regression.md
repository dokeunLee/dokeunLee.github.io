# 3장. 선형회귀와 최소제곱법

## 1. 선형회귀의 기본 형태

선형회귀는 입력 변수와 타깃 변수 사이의 관계를 직선 형태로 모델링하는 가장 기본적인 방법이다.

가장 단순한 형태는 다음과 같다.

```text
y = β₀ + β₁x + ε
```

여기서 각 기호의 의미는 다음과 같다.

| 기호 | 의미 |
|---|---|
| y | 타깃 변수, 예측 대상 |
| x | 입력 변수 |
| β₀ | 절편 |
| β₁ | 기울기 |
| ε | 오차 |

β₀와 β₁은 데이터로부터 학습하는 계수이고, ε은 모델로 설명되지 않는 오차를 의미한다.

---

## 2. 선형회귀로 알고 싶은 것

선형회귀의 목적은 단순히 예측만 하는 것이 아니다.

선형회귀를 통해 다음과 같은 질문을 다룰 수 있다.

- 입력 변수와 출력 변수가 관련이 있는가?
- 어떤 변수가 더 중요한가?
- 입력과 출력의 관계가 선형적인가?
- 변수들 사이에 시너지 효과가 있는가?
- 모델이 데이터를 얼마나 잘 설명하는가?

즉, 선형회귀는 예측 도구이면서 동시에 해석 도구다.

---

## 3. RSS와 최소제곱법

모델의 성능은 실제값과 예측값의 차이를 통해 평가한다.

선형회귀에서 대표적으로 사용하는 기준은 RSS, 즉 잔차제곱합이다.

```text
RSS = Σ(yᵢ - ŷᵢ)²
```

RSS가 작을수록 실제값과 예측값의 차이가 작으므로, 모델이 데이터를 더 잘 맞춘다고 볼 수 있다.

선형회귀에서 최적의 β₀, β₁은 RSS를 최소화하는 값으로 구한다.

이를 위해 RSS를 β₀와 β₁에 대해 미분하고, 최소점 조건을 만족하는 값을 찾는다.

이 방식이 최소제곱법이다.

---

## 4. 성능 평가지표

선형회귀의 성능을 평가할 때는 여러 지표를 사용할 수 있다.

| 지표 | 의미 |
|---|---|
| RSS | 잔차제곱합 |
| RMSE | 평균제곱근오차 |
| R² | 데이터 변동성 중 모델이 설명하는 비율 |

특히 R²는 모델이 데이터의 변동성을 얼마나 설명하는지를 나타낸다.

R²가 1에 가까울수록 모델이 데이터를 잘 설명한다고 볼 수 있다.

다만 R²가 높다고 해서 반드시 좋은 모델이라는 뜻은 아니다. 변수 선택, 모델 복잡도, 일반화 성능도 함께 고려해야 한다.

---

## 5. 회귀계수 해석 시 주의점

회귀계수는 입력 변수가 타깃 변수에 미치는 관계를 해석하는 데 사용될 수 있다.

하지만 회귀계수 해석에는 주의가 필요하다.

- 계수가 크다고 해서 반드시 인과관계가 강한 것은 아니다.
- 변수의 스케일이 다르면 계수 크기를 단순 비교하기 어렵다.
- 상관관계와 인과관계를 혼동하면 안 된다.

따라서 회귀계수는 데이터의 관계를 설명하는 단서이지, 그 자체로 인과관계를 증명하는 것은 아니다.

---

## 6. 왜 최소제곱법을 쓰는가

선형회귀에서 오차를 제곱하는 이유를 설명하기 위해 최대우도추정, 즉 MLE가 도입된다.

MLE는 다음 질문에 답하는 방법이다.

> 주어진 데이터가 가장 그럴듯하게 관측되도록 하는 파라미터는 무엇인가?

이를 위해 데이터가 i.i.d., 즉 독립적이고 동일한 분포를 따른다고 가정한다.

그리고 관측 데이터가 나타날 확률을 라이클리후드로 표현한 뒤, 이 값을 최대화하는 파라미터를 찾는다.

계산 편의를 위해 보통 로그 라이클리후드를 사용한다.

---

## 7. MLE 예시

### 동전 던지기 예시

동전 던지기에서 앞면이 나올 확률을 추정한다고 하자.

MLE 관점에서는 관측된 동전 던지기 결과가 가장 그럴듯하게 나올 확률 값을 찾는다.

이때 앞면 확률의 MLE는 결국 표본평균으로 나온다.

### 정규분포 예시

데이터가 정규분포를 따른다고 가정하면, 평균과 분산의 MLE도 구할 수 있다.

이 경우 평균의 MLE는 표본평균으로 나오고, 분산도 관측 데이터의 흩어짐을 바탕으로 추정된다.

---

## 8. MLE와 최소제곱법의 연결

선형회귀에서 오차가 다음 조건을 만족한다고 가정하자.

- 평균이 0
- 분산이 일정함
- 정규분포를 따름

이 조건에서 MLE로 파라미터를 추정하면, 결국 RSS를 최소화하는 해와 같아진다.

즉, 최소제곱법은 단순히 계산 편의를 위해 만든 방법이 아니다.

정규오차 가정 아래에서, 최소제곱법은 최대우도추정으로부터 자연스럽게 나온 결과다.

---

## 9. 행렬 표현과 정규방정식

선형회귀는 행렬로 쓰면 계산이 훨씬 간단해진다.

여러 개의 관측값과 여러 입력 변수를 하나의 행렬로 표현할 수 있고, 최적 해도 정규방정식 형태로 깔끔하게 표현된다.

절편항도 입력에 1을 추가한 변수처럼 포함시킬 수 있다.

```text
Xβ ≈ y
```

이렇게 쓰면 선형회귀 문제를 벡터와 행렬 연산으로 다룰 수 있다.

---

## 10. 추정량 관점

선형회귀에서 구한 추정량 β̂는 비편향 추정량이다.

비편향이라는 것은 평균적으로 보면 실제 β를 맞춘다는 뜻이다.

또한 데이터가 많아질수록 β̂의 분산이 줄어들어 추정 정확도가 높아진다.

즉, 표본 수가 증가하면 추정량이 더 안정적이 된다.

---

## 11. Bias-Variance 핵심

추정량의 오차는 보통 다음처럼 분해해서 생각할 수 있다.

```text
MSE = Variance + Bias²
```

좋은 추정량은 편향이 작고 분산도 작아야 한다.

하지만 실제로는 두 조건이 서로 충돌하는 경우가 많다.

이것이 편향-분산 트레이드오프다.

| 항목 | 의미 |
|---|---|
| Bias | 평균적으로 얼마나 빗나가는가 |
| Variance | 데이터가 바뀔 때 추정값이 얼마나 흔들리는가 |

편향이 작은 모델은 평균적으로 맞는 방향으로 예측하지만, 분산이 크면 데이터 변화에 민감할 수 있다.

반대로 편향을 조금 허용하면 분산을 크게 줄여 전체 예측 성능을 개선할 수도 있다.

---

## 12. BLUE와 가우스-마르코프 정리

선형회귀는 비편향 추정량이며, 특정 조건 아래에서 가장 분산이 작은 최적 비편향 추정량이다.

이를 BLUE라고 한다.

```text
BLUE = Best Linear Unbiased Estimator
```

이 성질은 가우스-마르코프 정리가 보장한다.

다만 BLUE라는 말이 항상 MSE 기준으로 최적이라는 뜻은 아니다.

경우에 따라서는 약간의 편향을 허용하고 분산을 크게 줄이는 모델이 전체 예측 성능에서는 더 좋을 수 있다.

---

## 정리

3장에서 잡아야 할 핵심은 다음과 같다.

1. 선형회귀는 입력과 타깃의 관계를 직선 형태로 모델링하는 기본 방법이다.
2. RSS를 최소화하는 방식으로 회귀계수를 추정한다.
3. 정규오차 가정 아래에서 최소제곱법은 MLE와 연결된다.
4. 행렬 표현을 사용하면 선형회귀를 더 간단하고 일반적으로 다룰 수 있다.
5. 선형회귀 추정량은 비편향이며, 가우스-마르코프 정리 아래 BLUE 성질을 갖는다.
6. 좋은 예측을 위해서는 Bias와 Variance의 균형을 고려해야 한다.
