# 1장. 머신러닝과 딥러닝의 흐름

## 1. 머신러닝의 위치

머신러닝은 이제 특정 전공만의 기술이 아니라, 다양한 분야에서 필요한 기본 역량으로 자리잡고 있다.

인간의 학문은 탐구 대상을 점점 넓혀왔다.

```text
물질
↓
생명
↓
인간
↓
사회
```

탐구 대상이 복잡해질수록, 복잡한 현상을 이해하기 위한 도구도 함께 발전해왔다.

---

## 2. 전통 과학에서 통계학으로

전통적인 과학은 관찰과 귀납적 추론을 통해 발전해왔다.

이후 산업혁명과 함께 복잡한 데이터를 다루기 위한 핵심 도구로 통계학이 중요해졌다.

통계학은 관찰된 데이터를 바탕으로 현상을 설명하고, 불확실성을 정량적으로 다루는 방법을 제공했다.

하지만 21세기 빅데이터 시대에는 기존 통계만으로는 감당하기 어려운 문제가 커졌다.

- 데이터의 양이 급격히 증가함
- 데이터의 차원이 커짐
- 이미지, 음성, 텍스트처럼 비정형 데이터가 중요해짐
- 사람이 직접 규칙을 설계하기 어려운 문제가 많아짐

이런 배경에서 머신러닝과 딥러닝이 실질적인 도구로 필요해졌다.

---

## 3. AI와 딥러닝의 역사

AI와 딥러닝은 갑자기 등장한 기술이 아니다.

대략적인 흐름은 다음과 같다.

```text
1940년대 수학적 모델
↓
퍼셉트론
↓
XOR 문제
↓
신경망 연구의 침체와 부활
↓
2012년 ImageNet 이후 딥러닝 시대
```

초기에는 인간의 뉴런을 수학적으로 모방하려는 시도가 있었다.

이후 퍼셉트론이 등장하면서 간단한 분류 문제를 풀 수 있게 되었지만, XOR 문제처럼 선형적으로 분리되지 않는 문제를 해결하기 어렵다는 한계가 드러났다.

이 한계는 신경망 연구의 침체로 이어졌지만, 이후 계산 자원, 데이터, 알고리즘이 발전하면서 신경망 연구가 다시 부활했다.

---

## 4. 2012년 ImageNet과 딥러닝 시대

딥러닝이 본격적으로 주목받은 계기 중 하나는 2012년 이미지 인식 대회 ImageNet이다.

이 대회에서 딥러닝 기반 모델이 기존 방법보다 큰 성능 향상을 보이면서, 딥러닝은 이미지 인식 분야의 핵심 방법으로 자리잡기 시작했다.

이후 딥러닝은 여러 분야에서 빠르게 발전했다.

- 이미지 인식
- 객체 탐지
- 음성 인식
- 자연어 처리

다만 딥러닝이 모든 문제를 해결한 것은 아니다.

여전히 다음과 같은 문제가 남아 있다.

- 많은 데이터 필요
- 높은 계산 비용
- 해석 가능성 부족
- 일반화 문제
- 학습 안정성 문제

## 5. 이 수업의 구성

이 수업은 머신러닝과 딥러닝의 흐름을 바탕으로, 통계적 머신러닝부터 현대 딥러닝 모델까지 다룬다.

수업 구성은 크게 두 부분으로 볼 수 있다.

| 구분 | 내용 |
|---|---|
| 전반부 | 통계적 머신러닝 |
| 후반부 | 딥러닝, CNN, RNN, Transformer |

단순히 모델 이름을 외우는 것이 아니라, 다음 내용을 함께 다루는 과정이다.

- 문제 접근법
- 이론적 배경
- 코드 작성
- 학습 시 주의사항
- 모델 선택과 평가

## 정리

1장에서 잡아야 할 핵심은 다음과 같다.

1. 머신러닝은 다양한 분야에서 필요한 기본 역량이 되었다.
2. 복잡한 데이터와 비정형 데이터를 다루기 위해 머신러닝과 딥러닝이 필요해졌다.
3. 딥러닝은 퍼셉트론, XOR 문제, 신경망 연구의 부활을 거쳐 발전했다.
4. 2012년 ImageNet 이후 딥러닝이 본격적으로 주목받기 시작했다.
5. 이 수업은 통계적 머신러닝과 딥러닝 모델을 함께 다루는 과정이다.
